# Roadmap

## Foundation

- Native solution, Windows CI, packaging identity, and locked runtime contract
- Framed App/Agent protocol and process-lifetime enforcement
- Virtualized native dual-pane performance gate

## Transfer workspace

- Connection profiles and session tabs
- Local/remote navigation and file operations
- Shared job activity, retry/resume/cancel, and bandwidth controls
- Parallel transfer and segmented-download policies

## LFTP workflows

- Upload/download mirror definitions and dry-run review
- Remote editing with remote-change conflict detection (implemented; real-server acceptance remains)
- Safe advanced console
- Remote-to-remote FXP/client-relay jobs
- Run-once schedules and optional tray lifetime

## Distribution

- Self-contained x64 MSIX
- App Installer on-launch/background/manual updates
- Unsigned CI artifacts and offline trusted-test signing
- Runtime inventory, license/source evidence, SBOM, and packaged smoke tests

## Deferred

- Recurring schedules
- Windows 10
- Native ARM64 runtime
- Portable ZIP distribution
- Additional LFTP protocol families
